프로젝트 소개

Java17, Spring boot 3.2 , Security 6.2,thymeleaf
Oracle19c, JPA, QueryDsl
Tool: IntelliJ, Git
docs: Adoc, REST-DOC
홈쇼핑 프로젝트의 목표로 개발하였고
보안,인증,카테고리,상품,결제,이벤트,쿠폰 이 주요 기능 이고
1인 개발이며 제작 기간은 1~2달 ,
CI/CD 는 Git 으로 진행 됩니다.
클린 아키텍처 구조를 참고해며 도메인 패키징을 했고
상위 계층의 객체를 사용하지 않게 설계 설계했습니다.
그리고 단위 테스트 방식으로 서비스 & API 테스트를
만들었고 Gitflow 방식으로 코드를 관리했습니다.
최소 비용으로 진행 되었으며 아래와 같습니다.
AWS: app , batch server [무료]
OCS : image , db(oracle) [무료]
sms 인증 : apick.com [건당 6원]
결제 : bootpay (테스트 용)[무료]
한 번에 보는 코드 요약
·Optional를 사용하여 쿼리 값을 미리 검증
·예외 처리는 조건에 맞지 않다면 ErrorCode 로 반환
·코드 2 Depth 이하 지향 & 일급 컬렉션 우선 사용 & 코드 컨벤션을 지키며 개발
·코드가 길어지거나 반복되면 메서드 분리
